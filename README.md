# HYPER XENO

Premium next-gen cybersecurity SaaS — built on React + TanStack Start + Lovable Cloud (Supabase).

## Run locally
```
npm install
npm run dev
```

## Deploy to Netlify
This repo includes `netlify.toml`. Just drag-drop the folder into Netlify or connect via Git. The `.env` file already contains your Lovable Cloud (Supabase) credentials, so the deployed site stays connected to the same database, auth, and admin account you've been using in Lovable.

Build command: `npm run build`  ·  Publish directory: `dist`
